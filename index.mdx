# SignalRoom docs home

Status: Current repo-native entry point. Public publishing should stay conservative until trial onboarding is ready.

SignalRoom is the public-facing name and landing layer. Warmr is the app, repo, and control-plane implementation.

SignalRoom is private software for TikTok operators. It is for teams that want to run their own TikTok phone setup with device lanes, posting runs, account state, logs, cooldowns, and operator approvals in one private cockpit.

The Warmr repo remains the source of truth. Mintlify should publish a curated mirror of these docs only after the project is pointed at this docs root and publication is approved.

## Safety framing

SignalRoom helps operators make attention workflows visible, paced, logged, and reviewable. It does not guarantee platform or account outcomes.
