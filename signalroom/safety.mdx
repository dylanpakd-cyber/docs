# Claims and safety

Status: Current copy guardrail.

SignalRoom docs should be clear, operational, and conservative.

Required safety line:

SignalRoom helps operators make attention workflows visible, paced, logged, and reviewable. It does not guarantee platform or account outcomes.

## Allowed framing

Current:

Use:

- Private software for TikTok operators.
- Built for founders and growth teams running their own phone setup.
- Runs on real iPhones owned by the customer.
- Built for Mac.
- Private license access.
- Guided onboarding.
- Call based access.
- Visibility, pacing, logs, receipts, review, readiness checks, and controlled runs.

## Do not imply

Avoid positioning SignalRoom as:

- phone hosting
- outsourced fleet operations
- an agency
- a broad enterprise social suite
- a product that promises platform outcomes
- a system for hiding or evading platform behavior

## Current copy rule

If a capability is verified in the repo, label it Current. If the repo shows the direction but not the finished operator feature, label it In progress. If it is roadmap language, label it Planned.
